Welcome to GitHub Pages.

This code provides three function with computing analytics over the Hawaii Directory of Green Employers.

  1. listIndustries(data). This function can be passed greenjobs and returns an array of strings indicating all the industries in the dataset (no duplicates).

  2. countyGreenJobs(data). This function can be passed greenjobs and returns an object where the keys are County names and the values are the number of Green Jobs listed in that County.

  3. jobswithKeyword(data, keyword). This function can be passed greenjobs and a string and returns a list of Job Titles containing the passed string.
